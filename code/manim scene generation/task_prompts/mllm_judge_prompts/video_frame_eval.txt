## Dimension: Visual Quality (Single Frame) (1–5)

### Evidence Requirements (MANDATORY)

* The comprehensive_evaluation MUST cite concrete, visible observations from THIS frame (e.g., “text edges are jagged/pixelated,” “contrast is low,” “banding in background gradient,” “fonts mismatch between title and labels”).

### Checklist (STRICT)

A) Style Coherence (within this frame)

* Fonts consistent (no obvious mixed font families/styles unless clearly intentional)
* Stroke widths consistent (lines/axes/diagrams not mismatched)
* Color usage coherent (no random clashing colors; consistent palette)
* Diagram styling matches text styling (labels and geometry/graphs feel like one system)

B) Readability / Sharpness

* Contrast adequate between text and background
* Text/math symbols legible at normal viewing (not tiny, not faint)
* No excessive blur; anti-aliasing acceptable (not overly jagged or smeared)

C) Still-frame Artifacts

* Compression artifacts: blockiness, ringing/halos, mosquito noise
* Banding in gradients, pixelation, smeared edges
* Any artifact that reduces legibility must be noted and penalized

### Score Caps (MANDATORY)

* If {video_frame} is missing/unreadable → **score = 1**
* If any key text is not readable due to blur/low resolution/low contrast → **score ≤ 2**
* If noticeable artifacts are present and affect readability (pixelation, ringing, banding) → **score ≤ 3**
* If style inconsistencies are obvious (mixed fonts, mismatched stroke widths, clashing colors) → **score ≤ 3**
* If only minor issues exist and readability is fully preserved → **score ≤ 4**
* Score **5** is allowed only when: crisp text + strong contrast + minimal artifacts + coherent styling (all clearly satisfied)

### Scoring Anchors (STRICT)

* 5: Crisp and clean; excellent contrast; no meaningful artifacts; coherent styling throughout.
* 4: Very good; tiny artifacts or slight softness, but everything remains comfortably readable.
* 3: Acceptable; noticeable softness/artifacts/style inconsistencies, yet still readable.
* 2: Poor; artifacts/blur/contrast issues significantly harm readability.
* 1: Unusable/unjudgeable frame (missing, corrupted, or cannot see content).
